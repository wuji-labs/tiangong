# 例 02 · 金融 App 配色(Color Palette · 随类赋彩 / 五行 / 节制)

锁定本 skill 在「选配色」场景下,如何用"色随物性 + 五行 + 节制"而非"调色盘随手挑"。

---

## INPUT(用户请求)

> 我们的理财 App 要定一套配色。设计师给了一版:主色亮蓝 + 渐变紫 + 橙色强调 + 绿涨红跌,还加了金色会员色。总感觉花、不够稳。帮我定。

## 没有本 skill 的产出(反模式)

```
蓝色显专业,紫色显高级,橙色抓眼球做 CTA,金色做会员尊贵感,
再加渐变让它更现代……
（五六种高饱和色并置,无主次,「五色令人目盲」,理财最忌花——用户感到的是慌,不是稳）
```

## 有本 skill 的产出(随类赋彩 + 先墨后色)

```
[天工 | 目标:理财 App 配色 | 性:钱——稳、可信、精确 | 五行:黑水(深、智)为基 + 一点赤火(涨) | 主色:墨 + 米黄底,天青担信任,克制 | 留白:留呼吸 | 一句神:让人把钱放这儿很安心]
```

### 1 格物(应物象形)— 先问"钱"之性
理财之性:**稳、可信、精确、不慌**。配色第一要务是"让人安心",不是"抓眼球"。
- 五行:取**黑水**(北方、冬、深、智、藏)为基调——水主智与藏,合"理财"。
- 忌:高饱和大面积 + 渐变 + 多强调色,这些制造的是兴奋/焦虑,与"稳"相反。

### 2 先墨后色(墨分五色)— 灰度先立
理财 App 90% 是数字与表格,层次必须先用灰度立稳:

| 用途 | 墨级 | 值 |
|------|------|-----|
| 关键数字/标题 | 焦 | #1A1A1A |
| 正文 | 重 | #4A4A4A |
| 次要信息 | 淡 | #999999 |
| 分隔/底纹 | 清 | #E5E5E5 |

### 3 随类赋彩 — 一主色担信任,色随物性
在灰度骨架上,只加**极少**的色,每一色都有职责:

| 角色 | 选色 | 意象/出处 | 职责 |
|------|------|----------|------|
| 底 | 米黄 #F5E6CC | 宋瓷温底,非冷白 | 暖、安、不刺眼 |
| 主 | 天青 #68B0AB | 「雨过天青云破处」 | 信任、品牌、主 CTA |
| 涨 | 朱砂 #C83C23(克制用) | 传统红 = 吉/进 | 仅用于涨/重要正向 |
| 跌 | 黛 #4A4266(冷收) | 而非刺眼亮绿 | 跌/收敛,不喧宾 |

**删掉**:渐变紫、橙强调、金会员色——皆为"炫"而非"服务"(大巧若拙)。会员尊贵感用**留白 + 一枚天青底款**表达,不靠金。

### 4 节制检验(五色令人目盲)
- 全局可见色 ≤ 3(米黄底 + 天青主 + 一个涨/跌色),其余全是墨分五色。
- 「色不过五,五色之变不可胜观」(《孙子兵法》借用):限色反而层次更丰。
- 终检:把界面转灰度,层次是否仍清晰?清晰 = 骨立住了,色只是赋上去的。

---

**要点**:配色不是"挑好看的颜色",是先问**此物之性**(钱要稳),先用墨分五色把骨立住,再让一主色担情感。
节制不是贫乏——限色到 2–3,反而比满屏彩虹更显贵、更可信。
