# 例 01 · 茶品牌落地页(Landing Page · 气韵 / 留白 / 卷轴展开)

锁定本 skill 在「做一个有东方气质的落地页」场景下的标准产出与细节级别。

---

## INPUT(用户请求)

> 帮我做一个高端茶品牌的官网落地页。别又是那种满屏卡片、绿底白字、看着像每个茶电商的样子。

## 没有本 skill 的产出(反模式)

```
好的,顶部一个 Hero 大图 + 大标题 + 立即购买按钮,
下面三列产品卡片,再来一行客户 logo,再来一个 FAQ 折叠……
配色:主绿 #2E7D32 + 辅金 #FFD700,渐变背景,圆角阴影卡片。
（按通用电商模板堆满,无视"茶"之本性,无留白,五色杂陈——看着像每个茶电商)
```

## 有本 skill 的产出(循四步法)

```
[天工 | 目标:高端茶品牌落地页 | 性:茶——静、温、不慌 | 五行:青木(生发)+白金(素) | 主色:月白 #D6E4E1 + 天青 #68B0AB,墨色担字 | 留白:约 60% | 一句神:让人未读先静下来]
```

### 1 格物(应物象形)
先读"茶"之性,不套模板。
- 茶之性:**静、温、不慌、回甘**。它不该喊"立即购买",该让人慢下来。
- 五行:青木(生发、健康)为主,佐白金(素、克制)。季节取**春**(春山淡冶而如笑,《林泉高致》)。
- 语境:高端 = 雅(《长物志》宁朴无巧)。用户心境:想被安顿,不想被推销。

### 2 立骨(经营位置)
- 基准单位 8px,字号/间距/留白皆其倍数(材份制,《营造法式》)。
- 版式取**长卷**(卷轴展开):一屏一景,纵向滚动如展画,而非一屏塞满。
- 留白约 60%——「当其无,有器之用」(道德经 11),空处即设计。

### 3 赋彩(随类赋彩·墨分五色)
先墨后色:
- 层次用墨分五色担——标题焦(#1A1A1A)、正文重(#666)、提示清(#CCC)。
- 仅一主一辅两色:月白底 #D6E4E1 + 天青点 #68B0AB。**不上金**(金红 = 俗,《长物志》)。
- 色起意象名:背景"月白",按钮"雨过天青"。

### 4 点睛(气韵生动·卷轴展开)
按起承转合布四屏(《书谱》起承转合 + 卷轴长卷):

| 屏 | 卷轴段 | 内容 | 节奏 |
|----|--------|------|------|
| 1 起 | 留白最大 | 一片月白 + 一行小字「一盏,慢下来」+ 一枚天青茶汤特写 | 疏 |
| 2 承 | 信息层入 | 茶之来处(山场、节气),水墨地图 | 密 |
| 3 转 | 意外一笔 | 一个"借景"动效:茶汤晕开如泼墨(《天工开物》丹青) | 疏 |
| 4 合 | 收束 | 唯一 CTA「取一盏」,居中而不居正(经营位置) | 密 |

点睛细节:茶汤特写 hover 时微微荡漾(微交互),其余克制——大巧若拙,删到无可删。

---

**要点**:东方气质不是贴龙纹堆红金,是**读懂"茶"之性**后,用留白、墨色层次与卷轴节奏让页面"静下来"。
顺物性而设是依乎天理;套电商模板堆满是逆纹理。
