> **天地有大美而不言** — 천지는 큰 아름다움을 지니되 그것을 말하지 않는다.

**당신의 AI는 절반의 팔레트로 디자인하고 있다. 나머지 절반을 건네주자.**

대부분의 AI 디자인 학습은 서구의 프레임워크——머티리얼 디자인, 바우하우스, 스위스 스타일——에 지배되고 있다. 깔끔하고, 미니멀하고, 모던하지만…… 흔히 혼이 없다.

**천공**(TianGong)은 5000년에 걸친 중국 미학의 지혜를 AI 디자인에 불어넣는다. 장식으로서가 아니라, 아름다움·공간·리듬·의미를 이해하기 위한 완결된 철학 체계로서.

## 문제

```
당신: "랜딩 페이지를 디자인해 줘"
천공 없는 AI: 완벽한 그리드. 완벽한 여백. 완벽한…… 무(無).
천공 있는 AI: 숨 쉬는 페이지. 물처럼 흐르는 페이지.
              빈 공간이 콘텐츠보다 더 크게 말하는 페이지.
              모든 색이 의미를 품은 페이지.
```

## AI에게 무엇을 가르치는가

### 🎨 육법(사혁 육법, 5세기)
현존하는 세계에서 가장 오래된 디자인 프레임워크:

| 법칙 | 중국어 | 의미 |
|-----------|---------|---------|
| 기운생동 | 气韵生动 | 디자인은 **살아 있다**고 느껴져야 한다 |
| 골법용필 | 骨法用笔 | 아름다움에는 **뼈대**가 필요하다 |
| 응물상형 | 应物象形 | 디자인하기 전에 먼저 이해하라 |
| 수류부채 | 随类赋彩 | 색 = 의미이지, 장식이 아니다 |
| 경영위치 | 经营位置 | 빈 공간이 곧 디자인이다 |
| 전이모사 | 传移模写 | 전통을 익히고, 그런 뒤 넘어서라 |

### 🧠 오지(五智) — 의사결정 프레임워크

| 지혜 | 출처 | 스스로에게 물어라 |
|--------|--------|-------------|
| 도법자연 | 도덕경 제25장 | 자연스럽게 느껴지는가? |
| 대교약졸 | 도덕경 제45장 | 과시인가, 봉사인가? |
| 기이재도 | 역경 | 그것은 어떤 의미를 담고 있는가? |
| 천인합일 | 역경 | 맥락과 조화를 이루는가? |
| 중용지도 | 중용 | 균형이 잡혀 있는가? |

### 🏺 재료의 지혜 — 4가지 전통에서

| 전통 | 가르치는 것 | 디자인 적용 |
|-----------|---------|-------------------|
| 书法 서예 | 리듬, 긴장, 흐름 | 타이포그래피, 시각적 리듬 |
| 水墨 수묵화 | 층, 비움, 생명 | 시각적 위계, 여백 |
| 园林 정원 | 발견, 프레이밍, 시퀀스 | 내비게이션, 점진적 공개 |
| 瓷器 자기 | 절제, 숨은 아름다움, 형태 | 색상 팔레트, 표면 디자인 |

## 동양, 서양을 만나다

천공은 서구의 디자인 지식을 대체하지 않는다. 그것을 **완성한다**.

| 서양 | ＋ 중국 | ＝ 완전함 |
|---------|-----------|------------|
| 바우하우스: 기능적 명료함 | 기운: 정신적 공명 | 작동하고 또한 숨 쉬는 디자인 |
| 스위스: 그리드의 규율 | 서예: 리듬 있는 생명 | 흐르는 구조 |
| 머티리얼 디자인: 일관성 | 송대 미학: 세련된 취향 | 일관되고 또한 품격 있는 디자인 |
| 황금비: 비례 | 여백: 의도된 비움 | 말하는 공간 |
