> **天地有大美而不言** — 天地は大いなる美を備えながら、それを語らない。

**あなたのAIは半分のパレットでデザインしている。もう半分を渡そう。**

AIデザインの学習は、その大半が西洋のフレームワーク——マテリアルデザイン、バウハウス、スイス・スタイル——に支配されている。クリーンで、ミニマルで、モダン……だが、しばしば魂を欠いている。

**天工**（TianGong）は、5000年にわたる中国美学の叡智をAIデザインに吹き込む。装飾としてではなく、美・空間・リズム・意味を理解するための完全な哲学体系として。

## 課題

```
あなた：「ランディングページをデザインして」
天工なしのAI：完璧なグリッド。完璧な余白。完璧な……無。
天工ありのAI：呼吸するページ。水のように流れるページ。
              余白がコンテンツよりも雄弁に語るページ。
              すべての色が意味を帯びるページ。
```

## AIに何を教えるか

### 🎨 六法（謝赫六法、5世紀）
現存する世界最古のデザイン・フレームワーク：

| 法則 | 中国語 | 意味 |
|-----------|---------|---------|
| 気韻生動 | 气韵生动 | デザインは**生きている**と感じさせよ |
| 骨法用筆 | 骨法用笔 | 美には**骨格**が要る |
| 応物象形 | 应物象形 | デザインの前に理解せよ |
| 随類賦彩 | 随类赋彩 | 色＝意味であり、装飾ではない |
| 経営位置 | 经营位置 | 余白こそがデザインである |
| 伝移模写 | 传移模写 | 伝統に学び、そして超えよ |

### 🧠 五智 — 意思決定フレームワーク

| 智 | 出典 | 自らに問え |
|--------|--------|-------------|
| 道法自然 | 道徳経 第25章 | 自然に感じられるか？ |
| 大巧若拙 | 道徳経 第45章 | 見せびらかしか、それとも奉仕か？ |
| 器以載道 | 易経 | それはどんな意味を担うのか？ |
| 天人合一 | 易経 | 文脈と調和しているか？ |
| 中庸之道 | 中庸 | 釣り合いが取れているか？ |

### 🏺 素材の叡智 — 4つの伝統より

| 伝統 | 教えるもの | デザインへの応用 |
|-----------|---------|-------------------|
| 书法 書道 | リズム、緊張、流れ | タイポグラフィ、視覚的リズム |
| 水墨 水墨画 | 層、空白、生命 | 視覚的階層、余白 |
| 园林 庭園 | 発見、フレーミング、シークエンス | ナビゲーション、段階的開示 |
| 瓷器 磁器 | 抑制、隠された美、形 | カラーパレット、サーフェスデザイン |

## 東洋、西洋と出会う

天工は西洋のデザイン知識を置き換えるものではない。それを**完成させる**のだ。

| 西洋 | ＋ 中国 | ＝ 完全 |
|---------|-----------|------------|
| バウハウス：機能的な明晰さ | 気韻：精神的な共鳴 | 機能し、なおかつ呼吸するデザイン |
| スイス：グリッドの規律 | 書道：リズムある生命 | 流れる構造 |
| マテリアルデザイン：一貫性 | 宋代美学：洗練された趣味 | 一貫し、なおかつ趣のあるデザイン |
| 黄金比：比例 | 留白：意図された余白 | 語る空間 |
